# Python 3

This code is a simple implementation of a console calculator. It's aimed at beginners to python programming.<br /><br />
The application supports the following operations:

> __+__ addition<br />
> __-__ subtraction<br />
> __/__ division<br />
> __*__ multiplication<br />
> __sqrt__ root extraction<br />
> __pow__ degree expansion<br />
> __sin__ sin calculation<br />
> __cos__ cos calculation<br />
> __tg__ tangent calculation<br />
> __ctg__ calculation of cotangent<br />
